# Università degli Studi di Napoli Parthenope
## Progetto d'esame Tecnologie Web : Speed Mensa

Questo progetto nasce per semplificare l'accesso ai servizi della mensa universitaria;
L'applicazione web, sviluppata con **Python Flask**, permette agli studenti di:

- Prenotrare i pasti in anticipo
- Effettuare pagamenti online in modo sicuro
- Consultare menu e disponibilità giornaliera
- Gestire le prenotazioni
